Development of an application for radiological monitoring, focused on data collection and analysis using the MEAN stack (MongoDB, Express, Angular, Node.js)
# Radiological Environmental Monitoring Web Application

## Description
Web application for radiological monitoring and data visualization.

## Tech Stack
- Angular 17
- Node.js
- Express.js
- MongoDB
- REST API

## Features
- User authentication
- Adding new samples
- Calculations for existing samples
- Editing existing samples
- Deleting existing samples
- Adding new tracking records
- Generating reports
- Sending reports via email
- Downloading reports

## How to run
Frontend:
npm install
ng serve

Backend:
npm run serve
